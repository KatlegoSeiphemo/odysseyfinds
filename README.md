# Odyssey Finds

Odyssey Finds is a curated e-commerce platform for high-quality products.

## Project Structure

- `frontend/`: React-based frontend using Tailwind CSS and Radix UI.
- `backend/`: Flask-based API for products, cart, and orders.

## Getting Started

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm start`

### Backend

1. `cd backend`
2. `pip install -r requirements.txt`
3. `python server.py`
